# 오라클 다운로드 

1. 검색 : 오라클
  https://www.oracle.com/kr/

2. 메뉴 > 제품
          > 하드웨어 및 소프트웨어
          > Oracle Database

3.  Oracle Database 19c 다운로드 버튼 클릭
             

# 오라클 설치

1. 오라클 설치압축파일.zip
  [개인폴더]>[SETUP] 폴더로 이동

2. 알아서 풀기 (압축해제)

3. 압축 해제 후, 폴더 안에
  setup.exe 실행


# 설치 진행
1. 단일 인스턴스 데이터 베이스 생성 및 구성

2. 서버 클래스

3. 표준 설치

4. 가상 계정 사용

5. Oracle Base 
   : C\개인폴더\SETUP\oracle
    (SETUP 폴더 안에 oracle 새 폴더 만들기)

6. 전역 데이터베이스 이름 : orcl

7. 시스템 계정 비밀번호 : 123456
